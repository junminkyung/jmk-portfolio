02 / APPROACH

실제 로봇에서
안정적으로 동작하는가?

01

현상을 수치로 바꿉니다.

감각적인 '떨림'이나 '느림'을 표준편차, 변동 범위, 처리 시간 분포로 정의합니다. 개선 전후를 같은 데이터로 비교할 수 없다면 개선이 아니라고 생각합니다.

02

실패 조건을 재현합니다.

가림, 원거리, 저조도, 연산 부하 같은 실패 상황을 실제 데이터와 시뮬레이션 양쪽에서 반복 가능한 시험으로 만듭니다.

03

정확도와 실시간성을 함께 봅니다.

검출 성능만 올리지 않고 ROI, 실행 조건, 센서 구성까지 시스템 전체 비용을 함께 최적화합니다.
